<h1>Fitur User Masyarakat</h1>

- Modul Edukasi Interaktif
    1. Panduan menanam (buah/sayur) dalam bentuk step-by-step.
    2. Tutorial membuat pupuk organik/kompos.
    3. Simulasi interaktif (misalnya: memilih jenis tanah → muncul rekomendasi tanaman).

- Perpustakaan Digital (Artikel & Video)
    1. Video pendek “how to”.
    2. Artikel singkat tentang lingkungan dan pertanian berkelanjutan.
    3. Fitur bookmark materi.

- Kuis & Tantangan Hijau (Gamifikasi)
    1. Kuis edukasi pertanian.
    2. Tantangan mingguan (misalnya “tanam bibit dalam 7 hari”).
    3. Sistem poin & badge untuk meningkatkan engagement.

- Forum Diskusi Warga
    1. Masyarakat bisa tanya-jawab dengan petani atau warga lain.
    2. Upvote, komentar, dan upload foto tanaman/bibit.

<h1>Fitur User Petani</h1>

- Pemindaian Tanah (Soil Scanner)
    1. Input data tanah manual / sensor / foto (jika menggunakan AI vision).
    2. Output berupa tingkat kesuburan, pH, kelembapan.
    3. Rekomendasi tanaman berdasarkan hasil analisis.

- Sistem Monitoring Tanaman
    1. Jadwal tanam, penyiraman, dan pemupukan.
    2. Notifikasi otomatis (reminder perawatan).
    3. Catatan kondisi tanaman harian (foto + deskripsi).

- Pencatatan Hasil Panen
    1. Input jumlah panen, kualitas, tanggal panen.
    2. Statistik perkembangan produktivitas lahan.
    3. Grafik panen bulanan/tahunan.

- Forum Petani
    1. Diskusi khusus antar petani.
    2. Tips budidaya dan solusi masalah hama.
    3. Ruang tanya-jawab dengan expert (opsional).

<h1>Fitur User Rt/Rw</h1>

- Dashboard Monitoring Pertanian
    1. Data jumlah petani aktif.
    2. Data aktivitas tanam dan hasil panen.
    3. Monitoring lahan kosong/lahan pertanian yang terdaftar.

- Manajemen Konten Edukasi
    1. RW/RT bisa mengupload materi edukasi (PDF, gambar, video).
    2. Fitur kategori (lingkungan, kebersihan, pertanian, dsb).
    3. Jadwal publikasi konten untuk masyarakat.

- Fitur Kolaborasi Pemasaran Hasil Panen
    1. Input harga dan ketersediaan hasil panen dari petani.
    2. Integrasi dengan marketplace lokal/dinas perdagangan.
    3. Notifikasi ke warga saat ada panen atau promo produk lokal.
